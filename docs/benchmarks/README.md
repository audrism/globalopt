# Benchmark Docs

This folder contains consolidated benchmark documentation.

- `COMPARISON_NARRATIVE.md`: benchmark narrative across Python and R interfaces
  (gap-to-optimum, runtime, memory footprint).
- `GAP_ASSESSMENT.md`: implementation coverage vs upstream routines and remaining gaps.
- `WINNER_TABLES.md`: per-problem winners and margin-to-second-best tables for
  Python and R comparison outputs.
- `APPLES_TO_APPLES.md`: side-by-side Python vs R comparison under the same
  translated globalopt method set and benchmark settings.

Generated benchmark tables and raw outputs remain under:

- `python/examples/output/`
- `r/globalopt/examples/output/`
